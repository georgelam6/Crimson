# Current Features
 - Basic 3D rendering
 - Scripting with AngelScript
 - Bare-bones editor
 - Scene serialisation

# Building
### Prerequisites
 - Premake5
 - A C++11 compiler
 - A GPU with OpenGL 4.2+

### Unix
```
./build.sh
./run.sh
```

### Windows
Generate Visual Studio/CodeLite project using Premake5, open the project and click "build".
